# ebert-haunt
Changes the byline of the modern reviews on [rogerebert.com](https://www.rogerebert.com) to "Roger Ebert's Ghost"

## Download
* [Firefox](https://addons.mozilla.org/en-US/firefox/addon/ebert-haunt/)
* Chrome (Coming soon)
